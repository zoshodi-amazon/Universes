## Session: Freeze v3.1.0

### Decisions

1. `Env/` is a canonical module directory — every module has `Env/{Inputs/,Outputs/}`
2. `Env/Inputs/` = IO boundary (secrets, cache, external data, API responses)
3. `Env/Outputs/` = persistent state (artifacts, logs, render_logs, docs/tracker)
4. ≤7 phases per module (excluding Pipeline) — new core invariant (#18)
5. Every module has its own `Env/Outputs/docs/tracker/{MM-DD-YY}.md`
6. `Prelude/Modules/` renamed to `Modules/` at repo root
7. The flake itself is the universe-level monad: `flake.nix` = inputs, `Modules/` = outputs. No `Env/` at repo root — the `Env/` convention applies to modules within the tree, not the universe root.
8. Canonical exemplar: `Modules/Labs/RL/` (7 phases + 1 Pipeline)
9. Moved tracker and definitions from root `Env/` into `Modules/User/Docs/`
